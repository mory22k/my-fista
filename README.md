# my-fista

A supersimple implementation of Fast Iterative Shrinkage-Thresholding Algorithm (FISTA) for LASSO.

This repository is intended for educational purposes and is not optimized for performance.

The implementation is mainly based on Amir Beck and Marc Teboulle, A Fast Iterative Shrinkage-Thresholding Algorithm for Linear Inverse Problems, SIAM Journal on Imaging Sciences 2009 2:1, 183-202 (2009). doi: <https://doi.org/10.1137/080716542>
